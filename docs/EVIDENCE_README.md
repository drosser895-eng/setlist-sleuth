# EVIDENCE PACK README

## Purpose
This package contains the artifacts for the Blaze Mix Master Revenue-First pilot staging verification. It provides cryptographic proof of score integrity and evidence of non-custodial payment infrastructure.

## Contents
- `/proofs/`: Merkle proof JSON bundles for individual matches.
- `/db/`: Database extracts showing anchored roots and transaction hashes.
- `harness_output.txt`: Logs from the attestation simulation.
- `REGULATOR_PACK.md`: Formal audit brief.
- `scripts/`: Tools used to generate and verify the evidence.

## Verification
Follow the steps in `REGULATOR_PACK.md` to re-verify any proof bundle using the included scripts.
